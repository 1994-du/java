# Spring Boot 项目接口文档

## 一、认证相关接口 (AuthController)

| 接口路径 | 请求方法 | 功能描述 | 请求参数 | 备注 |
|---------|---------|---------|---------|------|
| /api/auth/login | POST | 用户登录 | username: 用户名<br>password: 密码 | 需要验证 |
| /api/auth/register | POST | 用户注册 | username: 用户名<br>password: 密码<br>gender: 性别 | 密码长度不能少于6位 |
| /api/auth/reset-password | POST | 密码重置 | username: 用户名<br>newPassword: 新密码 | |
| /api/auth/logout | POST | 退出登录 | 无参数 | 清除token cookie |
| /api/auth/getMenus | POST | 获取当前用户菜单 | 通过cookie中的token获取 | 需要登录 |

## 通用响应格式

所有接口统一使用 ApiResponse 格式响应：

```json
{
  "success": true,
  "message": "操作成功",
  "data": { ... },
  "total": 100
}
```

## 说明

- 大部分需要认证的接口通过 Cookie 中的 token 进行身份验证
- WebSocket 接口支持实时通信
- 文件上传限制最大5MB
- 密码相关接口有长度限制（至少6位）
